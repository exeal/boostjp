graph グローバル変数¶

unspecified graph¶

グラフィカルな文字にマッチする。

説明¶

文字がグラフィカルな文字か決定する正規表現特性。グラフィカルな文字以外にマッチさせる場合は ~digit を使用する。

注釈

graph は Perl の /[[:graph:]]/ と等価である。~graph は Perl の /[[:^graph:]]/ と等価である。
