検索・置換書式化文字列の構文¶

  • sed 書式化文字列の構文
  • Perl 書式化文字列の構文
  • Boost 拡張書式化文字列の構文

書式化文字列は、アルゴリズム regex_replace および match_results::format で文字列を変換するのに使用する。

書式化文字列には sed 、Perl および Boost 拡張の 3 種類がある。

これとは別に、上に挙げた関数にフラグ format_literal を渡すと書式化文字列は直値文字列として扱われ、出力にそのままコピーされる。
