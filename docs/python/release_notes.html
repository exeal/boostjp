バージョン 1.67¶

  • Boost.Python のライブラリ名に Python のバージョンを接頭辞に含めるようにした。Python 2.7 でコンパイルすると、ライブラリ名は boost_python27 や boost_numpy27 などとなる。複数のバージョンの Python に対応したライブラリを一度に扱えるようにしたものだ
